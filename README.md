# Содержание:
1. start_programm.bat - батник для запуска
2. voice - код для запуска разпознователя голоса
3. robot - код для запуска робота

# Установка на другом компе:
1. Скопировать репозиторий
2. установить все библиотеки в локальную обоолчку .venv из voice/requirements.txt и robot/requirements.txt (на репозитории данная папка уже есть)

# Запуск программы:
1. Выполнить start_programm.bat. Два отдельных окна. Разпознователь речи можно растягивать, робот статический
